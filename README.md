# To-Do List Application

A clean Python-based To-Do List application developed as part of my internship at DecodeLabs. This project is designed to help users efficiently manage their daily tasks.

---

## Features
* Add task
* View task
* Remove task
* Mark task as done

---

## Tech Stack
* **Language:** Python 3.x

---

## How to Run the Application
1. Open your terminal or command prompt.
2. Make sure you are inside the project folder.
3. Run the application using the following command:
   python todo_list.py
